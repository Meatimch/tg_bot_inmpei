# Telegram Bot for INMPEI

Телеграм-бот, разработанный для автоматизации части процессов Института наставничества МЭИ.

## Возможности

* регистрация пользователей;
* хранение данных в SQLite;
* система FAQ;
* запись на интервью;
* рассылка уведомлений;
* выдача и сбор домашних заданий;
* экспорт данных в Google Sheets;
* квесты и активности для наставников.

## Технологии

* Python 3
* pyTelegramBotAPI
* SQLite
* Google Sheets API (gspread)

## Запуск

1. Установить зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Создать файл `creds.json` с учетными данными сервисного аккаунта Google.

3. Указать токен Telegram-бота в `main.py`.

4. Запустить приложение:

   ```bash
   python main.py
   ```

## Примечание

Проект разрабатывался как внутренний инструмент для организации работы Института наставничества и не предполагает использование в качестве универсального решения.
